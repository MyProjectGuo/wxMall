# 微信小程序商城

#### 项目介绍
微信小程序商城，小程序商城模板，简单实用的微信小程序商城

#### 软件架构
微信小程序结构


#### 安装教程

1. git clone https://gitee.com/dlsdls/wxmall-pub.git
2. 用微信小程序编辑器打开
3. 微信小程序编辑器预览
